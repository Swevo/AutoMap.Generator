Object mapping at
compile time.

Add [Map(typeof(OrderDto))] to your class. AutoMap.Generator writes the mapping code for you — at build time, with zero reflection, zero runtime overhead, and full AOT support.

$ dotnet add package AutoMap.Generator
your code
[Map(typeof(OrderDto))]
public class Order
{
    public int    Id       { get; set; }
    public string Customer { get; set; }
    public decimal Total   { get; set; }
}
generated at build time
// ✨ AutoMapExtensions.g.cs
public static OrderDto ToOrderDto(
    this Order src)
{
    return new OrderDto
    {
        Id       = src.Id,
        Customer = src.Customer,
        Total    = src.Total,
    };
}

Everything you need

Six versions of thoughtful features — from basic property mapping to enum-to-enum switch expressions.

🎛️

Conditional mapping

[MapWhen("src.IsActive")] wraps a property in a compile-time ternary. Optional Fallback for the false branch. Composes with [MapWith] and flattening.

🔄

Bidirectional mapping

[Map] on the source or [MapFrom] on the destination. Add Reverse = true to get both directions from one attribute.

🪆

Nested & collection mapping

When nested type has a [Map], it's resolved automatically. List<T>, T[] and other collections are mapped via LINQ — no configuration.

📐

Automatic flattening

Name your DTO property CustomerName and AutoMap.Generator resolves it as src.Customer?.Name by walking the source type tree — up to 3 levels.

🏗️

Constructor mapping

Positional records and primary-constructor classes are detected automatically. new OrderDto(src.Id, src.Name) — no attribute needed.

🔢

Enum mapping

Different enum types are mapped via compile-time switch expressions matched by name. Use [MapEnum] to rename individual values.

✍️

Custom expressions

[MapWith("src.Price.ToString(\"C2\")")] injects any C# expression verbatim. [MapDefault("\"N/A\"")] adds null coalescing.

🛡️

Build-time diagnostics

Six diagnostics (AM001–AM006) catch empty mappings, missing properties, unresolved types and unmatched enum values — all as IDE warnings/errors, never at runtime.

💉

IAutoMapper<T, R> interface

Every mapping gets a generated sealed mapper class with a static Instance. Register in DI for testability, zero allocation.

⚡

Zero runtime cost

No reflection. No expression compilation. No startup cost. Generated code is plain C# — the compiler optimises it like hand-written code. Fully AOT & MAUI compatible.

Examples

Everything you need to know — in code. 

// [Map] — attribute on the source class
[Map(typeof(OrderDto))]
public class Order
{
    public int     Id       { get; set; }
    public string  Customer { get; set; } = "";
    public decimal Total    { get; set; }
}

public class OrderDto
{
    public int     Id       { get; set; }
    public string  Customer { get; set; } = "";
    public decimal Total    { get; set; }
}

// Usage (generated extension method):
var dto = order.ToOrderDto();

// [MapFrom] — attribute on the destination class (keeps source clean)
[MapFrom(typeof(Order))]
public class OrderDto { ... }

// Reverse mapping — both directions from one attribute:
[Map(typeof(OrderDto), Reverse = true)]
public class Order { ... }
// order.ToOrderDto()  +  dto.ToOrder()  ← both generated
public class Order
{
    public int    Id           { get; set; }
    public string Customer     { get; set; } = "";
    public string InternalCode { get; set; } = "";  // excluded
    public string ClientRef    { get; set; } = "";  // renamed
}

[MapFrom(typeof(Order))]
public class OrderDto
{
    public int    Id       { get; set; }

    [MapIgnore]                     // exclude this property
    public string InternalCode { get; set; } = "";

    [MapProperty("ClientRef")]      // map from differently-named source property
    public string Reference { get; set; } = "";

    [MapWith("src.Customer.ToUpperInvariant()")]  // custom expression
    public string CustomerUpper { get; set; } = "";

    [MapDefault("\"N/A\"")]         // null substitution
    public string Customer { get; set; } = "";
}
public class Order
{
    public bool    IsActive  { get; set; }
    public bool    IsPremium { get; set; }
    public string  Name      { get; set; } = "";
    public decimal Price     { get; set; }
    public Customer? Customer { get; set; }
}

[MapFrom(typeof(Order))]
public class OrderDto
{
    // Map only when condition is true; false branch = default
    [MapWhen("src.IsActive")]
    public string Name { get; set; } = "";
    // Generated: Name = src.IsActive ? src.Name : default,

    // Custom fallback for the false branch
    [MapWhen("src.IsPremium", Fallback = "\"Standard\"")]
    public string Tier { get; set; } = "";
    // Generated: Tier = src.IsPremium ? src.Tier : "Standard",

    // Combine with [MapWith] — the custom expression becomes the true branch
    [MapWhen("src.IsActive")]
    [MapWith("src.Price.ToString(\"C2\")")]
    public string PriceLabel { get; set; } = "";
    // Generated: PriceLabel = src.IsActive ? src.Price.ToString("C2") : default,

    // Combine with flattening — flattened path becomes the true branch
    [MapWhen("src.IsActive", Fallback = "\"Anonymous\"")]
    public string CustomerName { get; set; } = "";
    // Generated: CustomerName = src.IsActive ? src.Customer?.Name : "Anonymous",
}
// Source: Order → Customer → Address → City
public class Address  { public string City  { get; set; } = ""; }
public class Customer { public string Name  { get; set; } = ""; public Address? Address { get; set; } }
public class Order    { public Customer? Customer { get; set; } }

// DTO names trigger automatic flattening — no configuration needed:
[MapFrom(typeof(Order))]
public class OrderDto
{
    public string CustomerName        { get; set; } = ""; // → src.Customer?.Name
    public string CustomerAddressCity { get; set; } = ""; // → src.Customer?.Address?.City

    [MapDefault("\"Guest\"")]
    public string CustomerNameFallback { get; set; } = ""; // → src.Customer?.Name ?? "Guest"
}

// Generated:
// CustomerName        = src.Customer?.Name,
// CustomerAddressCity = src.Customer?.Address?.City,
// Nested objects: mark each type with [Map]
[Map(typeof(AddressDto))]
public class Address { public string City { get; set; } = ""; }
public class AddressDto { public string City { get; set; } = ""; }

[Map(typeof(OrderDto))]
public class Order
{
    public int      Id      { get; set; }
    public Address? Address { get; set; }
}
public class OrderDto
{
    public int         Id      { get; set; }
    public AddressDto? Address { get; set; }  // ← resolved automatically
}

// Collections: same idea, any IEnumerable or T[] supported
[Map(typeof(ItemDto))]
public class Item { public string Sku { get; set; } = ""; }

[Map(typeof(OrderDto2))]
public class Order2
{
    public List<Item> Items { get; set; } = new();
}
public class OrderDto2
{
    public List<ItemDto> Items { get; set; } = new();
    // Generated: Items = src.Items?.Select(x => x.ToItemDto()).ToList()
}
// Positional records — detected automatically (no parameterless ctor)
public record OrderDto(int Id, string Customer);

[Map(typeof(OrderDto))]
public class Order { public int Id { get; set; } public string Customer { get; set; } = ""; }

// Generated:
// return new global::MyApp.OrderDto(src.Id, src.Customer);

// [MapConstructor] — force ctor mapping even with a parameterless ctor
[MapConstructor]
public class OrderDto2
{
    public int    Id       { get; }
    public string Customer { get; set; } = "";  // init prop — added to initializer block

    public OrderDto2() { }
    public OrderDto2(int id) { Id = id; }       // ← selected (longest)
}

// Generated:
// return new OrderDto2(src.Id) { Customer = src.Customer };
// Different enum types — mapped by name automatically
public enum OrderStatus    { Pending, Active, Cancelled }
public enum OrderStatusDto { Pending, Active, Cancelled }

// Generated switch expression:
// Status = src.Status switch
// {
//     global::MyApp.OrderStatus.Pending   => global::MyApp.OrderStatusDto.Pending,
//     global::MyApp.OrderStatus.Active    => global::MyApp.OrderStatusDto.Active,
//     global::MyApp.OrderStatus.Cancelled => global::MyApp.OrderStatusDto.Cancelled,
//     _ => default
// },

// [MapEnum] — rename a value
public enum SrcStatus
{
    [MapEnum("Running")]  // ← maps to DstStatus.Running
    Active,
    Done
}
public enum DstStatus { Running, Done }

// Same enum type → mapped directly, no switch needed
public class OrderDto { public OrderStatus Status { get; set; } }
// Status = src.Status
public class Order
{
    public int      Id     { get; set; }
    public decimal  Price  { get; set; }
    public string?  Region { get; set; }
    public List<string> Tags { get; set; } = new();
}

[MapFrom(typeof(Order))]
public class OrderDto
{
    public int Id { get; set; }

    // Custom C# expression — use 'src' to reference the source
    [MapWith("src.Price.ToString(\"C2\")")]
    public string PriceFormatted { get; set; } = "";

    // Null substitution — appended as ?? expr
    [MapDefault("\"Global\"")]
    public string Region { get; set; } = "";

    // Any valid C# expression
    [MapWith("src.Id > 1000 ? \"Premium\" : \"Standard\"")]
    public string Tier { get; set; } = "";

    [MapWith("string.Join(\", \", src.Tags)")]
    public string TagsJoined { get; set; } = "";
}

// Generated:
// PriceFormatted = src.Price.ToString("C2"),
// Region         = src.Region ?? "Global",
// Tier           = src.Id > 1000 ? "Premium" : "Standard",
// TagsJoined     = string.Join(", ", src.Tags),
// IAutoMapper<TSource, TResult> is emitted into your compilation.
// A sealed mapper class is generated for every mapping:

public sealed class OrderToOrderDtoMapper
    : IAutoMapper<Order, OrderDto>
{
    public static readonly OrderToOrderDtoMapper Instance = new();
    public OrderDto Map(Order source) => source.ToOrderDto();
}

// Use the static instance (zero allocations):
var dto = AutoMapExtensions.OrderToOrderDtoMapper.Instance.Map(order);

// Register in DI (Microsoft.Extensions.DependencyInjection):
services.AddSingleton<IAutoMapper<Order, OrderDto>>(
    AutoMapExtensions.OrderToOrderDtoMapper.Instance);

// Inject into services:
public class OrderService(IAutoMapper<Order, OrderDto> mapper)
{
    public OrderDto GetDto(Order order) => mapper.Map(order);
}

Build-time diagnostics

Problems surfaced as compiler warnings and errors — never at runtime.

ID Severity Description
AM001 ⚠ Warning No properties matched — mapping would be empty
AM002 ❌ Error [MapProperty("X")] references a non-existent source property
AM003 ❌ Error Type passed to [Map] / [MapFrom] could not be resolved
AM004 ⚠ Warning Destination property skipped — incompatible types, no registered mapping
AM005 ⚠ Warning Constructor parameter has no matching source property — default emitted
AM006 ⚠ Warning Source enum member has no matching destination member — _ => default fallback used

vs AutoMapper

AutoMap.Generator isn't trying to replace AutoMapper feature-for-feature. It targets the large majority of projects that use AutoMapper only for property mapping — and gives them something better.

🏭 AutoMapper

  • ✗ Reflection at startup
  • ✗ Runtime configuration validation
  • ✗ Not AOT / MAUI safe
  • ✗ Errors at runtime
  • ✓ Runtime conditional mapping
  • ✓ IQueryable projection
  • ✓ Polymorphic mapping

⚡ AutoMap.Generator

  • ✓ Zero reflection
  • ✓ Build-time validation (AM001–AM006)
  • ✓ AOT & MAUI safe
  • ✓ Errors at compile time
  • ✓ Plain C# — debuggable, readable
  • ✓ No DI setup required
  • ✓ netstandard2.0 — works everywhere

Migrate from AutoMapper

The most common AutoMapper patterns map directly to attributes. Here's a quick reference. See the full MIGRATION.md for a step-by-step guide.

AutoMapper AutoMap.Generator
CreateMap<Order, OrderDto>() [Map(typeof(OrderDto))]
.ReverseMap() [Map(..., Reverse = true)]
ForMember(d => d.X, opt => opt.Ignore()) [MapIgnore] on dest property
ForMember(d => d.X, opt => opt.MapFrom(s => s.Y)) [MapProperty("Y")] on dest property
ForMember(d => d.X, opt => opt.MapFrom(s => expr)) [MapWith("expr")] on dest property
ForMember(d => d.X, opt => opt.NullSubstitute("v")) [MapDefault("\"v\"")] on dest property
ForMember(d => d.X, opt => opt.Condition(s => s.Flag)) [MapWhen("src.Flag")] on dest property
Flattening (CustomerName → Customer.Name) Automatic — no config needed
Enum mapping Automatic switch expr + [MapEnum]
ConstructUsing(s => new Dto(s.X)) Automatic for records / [MapConstructor]
mapper.Map<OrderDto>(order) order.ToOrderDto()
IMapper (DI) IAutoMapper<Order, OrderDto> (optional)
